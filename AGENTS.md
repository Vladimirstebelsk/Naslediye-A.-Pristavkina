# AGENTS.md

Instructions for Codex and other coding agents working on this repository.

## Project Scope

This repository is a lightweight static school literature website about Anatoly Pristavkin and the story "Ночевала тучка золотая".

Use plain HTML and CSS only.

Do not add:

- React
- Vite
- Tailwind
- Bootstrap
- npm
- build tools
- external dependencies

Keep the site static, small, and easy to host on Netlify.

## Content Rules

- Preserve Russian content unless the teacher or repository owner explicitly asks for text changes.
- Do not publish private chat screenshots.
- Do not publish private names of students.
- Remove student names from public materials if requested by the teacher.
- Do not add placeholder content that looks final.

## Styling Rules

- Put shared styles in `style.css`.
- Avoid inline CSS in HTML pages.
- Keep page structure simple and readable.
- Do not redesign the site unless the task explicitly asks for design work.

Avoid generic AI-looking design patterns, including:

- glassmorphism
- neon gradients
- SaaS landing-page sections
- SaaS-style hero sections
- decorative filler sections
- meaningless decorative badges
- generic landing-page copy
- fake marketing phrases
- unrelated stock-style visuals

The design direction should be calm, literary, archive-like, readable, and suitable for a school literature project.

Every visible label, badge, heading, and card must communicate useful information. If a design element does not add meaning, remove it.

Homepage design must prioritize clarity, literature/history context, and atmosphere over decoration. It should immediately help visitors understand who Anatoly Pristavkin was, why "Ночевала тучка золотая" matters, and where to find biography, works, historical context, and analysis materials.

## Change Process

- Do not commit directly to `main`.
- Use a feature branch and open a pull request for changes.
- Keep pull requests narrowly scoped.
- For documentation-only setup tasks, do not modify the visual design.
